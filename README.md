# 🏠 Smart Home Automation System

A web-based IoT platform to remotely control and monitor home electrical devices with real-time automation.

## ✨ Features

- 🔌 Remote device control (lights, fans, ACs)
- 📊 Real-time power consumption monitoring  
- 🤖 Automated scheduling and routines
- 🎤 Voice control integration
- 📱 Responsive mobile-friendly interface
- 🔐 Secure user authentication

## 🛠️ Tech Stack

- **Backend:** PHP (Core)
- **Database:** MySQL  
- **Frontend:** HTML, CSS, JavaScript, AJAX
- **IoT Communication:** REST APIs

## 🚀 Installation

1. Clone repository: `git clone https://github.com/laibadilshad55-sys/smart_home.git`
2. Move to `C:/xampp/htdocs/`
3. Create database `smart_home_db`
4. Configure `db.php` with your credentials
5. Run at `http://localhost/smart_home`
